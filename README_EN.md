A library for generating Terraria 1.4 (PE) fonts.

## Code Example

[Example](./Example/Program.cs)

## FontGeneratorCLI (Command-Line Tool)

An interactive command-line tool for generating font resources specifically for **Terraria 1.4 PE**.

### Download & Installation

1. Download FontGeneratorCLI: [[Release]](https://github.com/UnrealMultiple/TRPEFontGen/releases/latest)
2. Install the .NET 10 Runtime: [[.NET 10]](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
3. Run `FontGeneratorCLI.exe`

> [!NOTE]
> The CI workflow only builds release binaries for certain platforms. For other platforms, please build from source.
