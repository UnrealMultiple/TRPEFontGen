## FontGeneratorCLI (命令行工具)

生成 **Terraria 1.4 PE**专用的字体资源的交互式命令行工具

### 下载 & 安装

1. 下载FontGeneratorCLI: [[Release]](https://github.com/UnrealMultiple/TRPEFontGen/releases/latest)
2. 安装.NET 10 运行时: [[.NET 10]](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0)
3. 启动`FontGeneratorCLI.exe`

> [!NOTE]
> 工作流没有构建其他平台的发布文件，如需使用请自行构建
